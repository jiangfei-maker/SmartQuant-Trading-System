# 个股信息查询模块

## 项目概述

这是一个功能完善的个股信息查询模块，用于获取和分析股票市场数据。该模块整合了多个数据源，提供全面的股票信息，包括基本信息、实时行情、技术指标、财务数据和基本面分析等功能。

## 架构设计

### 核心组件

1. **StockInfoQuery** - 主要查询类，整合多维度数据获取与分析功能
2. **DataProcessor** - 数据处理类，负责数据清洗和转换
3. **TechnicalIndicators** - 技术指标计算类，提供多种技术分析指标

### 数据流设计

```
客户端请求 → StockInfoQuery → 数据源接口 → 数据处理 → 结果返回
                            ↓
                      本地缓存机制
```

## 功能特点

### 数据获取功能

- **实时行情**：获取股票当前价格、涨跌幅、成交量等实时数据
- **基本信息**：获取公司简介、所属行业、主营业务等基本信息
- **技术指标**：计算MACD、RSI、KDJ、布林带、CCI、威廉指标等多种技术指标
- **财务数据**：获取公司财务报表数据和关键财务指标
- **基本面分析**：获取公司基本面数据，如市盈率、市净率等
- **股东结构**：获取公司股东结构和持股情况
- **公司新闻**：获取最新的公司相关新闻
- **分红信息**：获取公司历史分红情况
- **股东变动**：获取最新股东变动记录

### 性能优化

- **缓存机制**：实现了高效的内存缓存系统，减少重复请求
- **缓存过期**：支持设置缓存过期时间，默认5分钟
- **失败缓存**：对失败的请求结果也进行缓存，避免频繁请求不可用的API

### 错误处理

- **异常捕获**：每个方法都有完整的异常处理机制
- **备选数据源**：当主数据源不可用时，自动切换到备选数据源
- **模拟数据**：当所有数据源都不可用时，返回模拟数据确保系统稳定运行

## 技术指标说明

模块支持计算以下技术指标：

1. **MACD** - 异同移动平均线
2. **RSI** - 相对强弱指标
3. **KDJ** - 随机指标
4. **布林带** - 波动率指标
5. **MA** - 移动平均线（5日、10日、20日、60日等）
6. **量比** - 衡量相对成交量的指标
7. **换手率** - 股票交易活跃程度的指标
8. **OBV** - 能量潮指标
9. **CCI** - 顺势指标
10. **威廉指标(W%R)** - 超买超卖指标

## 使用方法

### 基本使用

```python
from 查询.stock_info_query import get_stock_info_query

# 获取查询实例
stock_query = get_stock_info_query()

# 查询个股完整信息
stock_code = "600000"  # 假设查询浦发银行
stock_info = stock_query.get_stock_complete_info(stock_code)

# 访问不同类型的信息
basic_info = stock_info['basic_info']
realtime_quote = stock_info['realtime_quote']
technical_indicators = stock_info['technical_indicators']
```

### 自定义缓存时间

```python
from 查询.stock_info_query import StockInfoQuery

# 创建实例时设置缓存时间为10分钟
stock_query = StockInfoQuery(cache_ttl=600)
```

### 清除缓存

```python
# 清除所有缓存数据
stock_query._clear_cache()
```

## 注意事项

1. 本模块依赖网络连接获取数据，请确保网络通畅
2. 部分数据源可能有访问频率限制，使用缓存机制可以有效减少API调用
3. 实时行情数据可能存在延迟，具体取决于数据源更新频率
